This article is for the book keeping of information used to configure the Zense VM.

Important packages installed

  • etckeeper -> To keep track of configuration changes
  • docker
  • docker-compose
  • nginx passenger -> To run rails apps directly without containers if needed

Deployment architecture

The deployment model had to support two use cases : 1, when containerised version of an app is available, 2 , when containerised version of app is not available.

Internet -> nginx-passenger running on host machine -> nginx-containerised (Will be used to reverse proxy to containers)
