If you have been following Zense then you must remember how did the old website look. Yes we know it was bad and that's why we always wanted to move out away from it. So after a few failed projects, we have finally managed to put together a basic portal. This is still a work in progress but we have planned a lot of things and hopefully everything turns out well.

Some things to look out for in the new website (Many features are still pending):

  • Projects section : It is for showcasing the work that we have done.
  • Members section: Finally a section which contains profile for all our members and contributors.

We are actively looking to make this website better, this is still work in progress!
